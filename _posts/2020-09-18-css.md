## css
- 含义：样式表
    - 层叠样式表

- 引入方式
    - `link`标签 -----外联
    - `style` 在head标签中引入style标签 -----内嵌
        - style属性: style属性值就是css代码 -----内联（一般不使用，）

- 书写规范
    - 选择器 你要为页面中的那一块设置样式
        - 基础选择器
            - `*` 通配符选择器
            - `.className` 类选择器 给标签添加一个class属性
            - `#idName` id选择器 给标签添加一个id属性 同一个id名在同一个页面中只可以出现一次（规范）,实际上可以同时出现
        - 基础选择器的优先级 id > class > tag > *
    - 样式书写规范

```css
* {
    属性名：属性值
}

div{
    属性名： 属性值
}

.table {
    属性名： 属性值
}
```
